# MAX6675 Flipper Driver

## Подключение

MAX6675 подключается к Flipper Zero по SPI:

| MAX6675 | Flipper GPIO |
|--------|--------------|
| VCC    | 3.3V (9)         |
| GND    | GND (любой)  |
| SCK    | B3 (5) или любой свободный цифровой     |
| CS     | B2 (6)     |
| SO     | A6 (3)         |

> MOSI не используется

---

```c
get_temperature() // Возвращает температуру в градусах Цельсия.
//-1.0f если ошибка датчика (термопара не подключена или обрыв CS)
